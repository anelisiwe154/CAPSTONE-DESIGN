## Description
This ERD depicts the relationships between events, alerts, and SOC analysts. Events generated by honeypots or MCP-driven attacks are correlated into alerts, which analysts review and respond to.

```mermaid
erDiagram
    EVENTS {
        int event_id PK
        string source_ip
        string destination_ip
        string attack_type
        datetime timestamp
    }
    ALERTS {
        int alert_id PK
        int event_id FK
        string severity
        string detection_rule
    }
    USERS {
        int user_id PK
        string analyst_name
        string role
    }
    EVENTS ||--o{ ALERTS : generates
    USERS ||--o{ ALERTS : reviews

```
